\subsection{File header}

The beginning 16 bytes of the file are byte order agnostic.
The first byte must have the value $(7\text{F})_{16}$ (127) followed by
$(45)_{16}$, $(4\text{C})_{16}$, and $(46)_{16}$ (ELF in ASCII encoding).
